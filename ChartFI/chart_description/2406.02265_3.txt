This heatmap visualizes the input attribution for each generated token (y-axis), where the attribution score of each input token (X-axis) is computed at each generation step. The brighter the color, the greater the attribution from the input token, with bright color cells corresponding to high attribution to the input token against a background of cells that are mostly near zero. A clear diagonal band emerges where matching words reinforce each other, particularly near the end of the sequence, where the alignment intensifies and becomes more concentrated. Within this pattern, high attribution scores are observed at “umbrella”, “boy”, “cattle”, and “over”, indicating the model is confidently mapping the core subject and objects across the sentences. Earlier in the sequence, the signal is weaker and more diffuse, suggesting less certainty or distributed attention around descriptive modifiers rather than locking onto a single counterpart. However, distinct anomalies exist within the data; negative scores are observed at the token “her” when the model is predicting the token “boy” and at the token “small” when predicting “herd”. These negative dips hint at mild confusion or the suppression of competing phrasing between adjacent tokens, contrasting with the otherwise high semantic agreement and clean correspondence found as the caption converges on concrete nouns.
