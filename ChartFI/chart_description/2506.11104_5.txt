This chart illustrates the retrieval accuracy on LongEval for LLaMA 3.2 3B and 1B models, comparing the performance of a full attention baseline (LLaMA) with the DAM method across context lengths from 3k to 39k. A key finding is that DAM consistently aligns with the dense model’s performance, as it closely matches the dense model’s retrieval accuracy across all evaluated sequence lengths and keyword positions. The comparison between model sizes, however, reveals significant differences in robustness. The larger LLaMA 3.2 3B model maintains relatively stable and high retrieval accuracy across most context lengths, with performance degradation appearing as localized, scattered pockets of low scores rather than a systemic failure. In contrast, the smaller LLaMA 3.2 1B model's performance is more volatile and sensitive to context length, exhibiting a more pronounced degradation with larger, contiguous areas of low accuracy emerging, particularly in contexts exceeding 25k tokens. Furthermore, for both models, performance varies based on the target keyword’s position within the sequence, even at fixed token lengths; the pattern is not a simple gradient from beginning to end but rather an interaction between context length and specific positional ranges, creating intermittent failure zones that become more frequent and severe for the 1B model.
