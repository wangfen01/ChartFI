The figure compares model performance across multiple classifiers and metrics while elucidating the feature types driving these predictions through attribution analysis. In the circular performance comparison, the multimodal feature R_TC from FMM_TC consistently outperforms unimodal features from other feature extractors with linear classifiers on drug-agnostic response prediction. Specifically, the fused representation R_TC yields the most robust gains, extending farther outward across metrics like AUROC, F1, BACC, and MCC, compared to raw features (X_T, X_C), PCA-reduced variants, or the unimodal BrainLM(X_T) representation. This performance advantage is contextualized by feature importance analysis via integrated gradients (IG), which illustrates how FMM_TC’s cross-dataset dynamic behaviors enhance adaptability. In the attribution plots, where the light blue background represents IG values for time-series feature R_T and the light red region denotes functional connectivity feature R_C, distinct dataset-dependent patterns emerge. For the OpenNeuro dataset, attribution magnitudes are significantly higher for time-series features, with the mean IG value dropping sharply when transitioning to the connectivity region, aligning with scenarios where time-series representations carry the primary discriminative signal. Conversely, in the in-house setting, this pattern reverses: functional connectivity attributions rise markedly above time-series values, indicated by a clear step-up in the mean IG magnitude within the red region. This contrast demonstrates that FMM_TC flexibly prioritizes the most effective modality in predictions, shifting its focus between temporal dynamics and connectivity structure depending on which feature type offers the strongest signal for a given cohort.
